# AI Visibility GEO SaaS

Bu proje, markaların AI cevap motorlarında görünürlüğünü ölçmek ve rakiplerine göre analiz etmek için geliştirilen modüler bir micro-SaaS projesidir.

## İlk hedef

Türkiye’deki küçük ve orta ölçekli e-ticaret markaları için AI görünürlük audit raporu oluşturmak.

## İlk modüller

- Ürün tanımı
- Rakip araştırması
- Örnek audit raporu
- Marka görünürlük analizi
- Rakip karşılaştırması
- Prompt bazlı raporlama
